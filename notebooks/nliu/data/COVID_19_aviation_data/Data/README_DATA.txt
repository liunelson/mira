This folder "Data" contains three subfolders " COVID_outputs", "Fight_outputs", "SEIR_sim_outputs". You can re-generate these files from the scripts in " Pre_processing" and "Code".


"COVID_outputs" are output files from  "Pre_processing/COVID_processing.m" It includes the  confirmed cases, deaths cases, recovered cases normalized by population (cases/person).


"Flight_outputs" are output files from  "Pre_processing/Flight_processing.m"
It contains the list of research flights and Air network adjacency matrix.


"SEIR_sim_outputs" are output files from  "Code/main.m".
It includes the county adjacency matrix, simulated infected level and output figures shown in the paper Section V.
